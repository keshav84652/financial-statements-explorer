# Financial Statements Explorer

An interactive educational tool for exploring the relationship between financial statements. This application visualizes how transactions and business events affect the Balance Sheet, Income Statement, and Cash Flow Statement simultaneously.

## Features

- **Interactive Financial Statements**: Real-time updates across all three primary financial statements
- **Transaction Simulator**: See how business events impact financial statements
- **Visual Flow Diagrams**: Understand how money flows between statements
- **Financial Ratio Analysis**: Interactive visualization of key financial ratios
- **Scenario Planning**: Test different business strategies
- **Educational Components**: Learn accounting concepts through interactive tutorials
